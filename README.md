## Setup

clone the repository by doing the following:

<code>git clone https://github.com/Hammad-Anjum/MealNest.git
cd MealNest
</code>

start the server with the following:

<code>cd API
npm i nodemon
nodemon server.js</code>

start the frontend with the following:

<code>cd Client
npm i
npm run dev</code>
